# The Food Spot - Restaurant Website

A modern, responsive restaurant website showcasing our menu, story, and online reservation system.

## Features

- Responsive navigation with sticky header
- Hero section with call-to-action
- About Us section
- Menu showcase with categories
  - Appetizers
  - Main Courses
- Online reservation system
- Contact information and social media links

## Technologies Used

- HTML5
- CSS3
- Responsive Design
- Flexbox & Grid Layout

## Project Structure

```
SNWProject/
│
├── index.html      # Main HTML file
├── style.css       # Stylesheet
└── README.md       # Documentation
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through the website sections
